Snake Case

Wörter mit "_" getrennt.

``` // Variables var wifi_pre_shared_key = "Hello"; // Functions function array_merge(arr1, arr2) { ... } ```

z.B. C++, C, Rust, PHP, Ruby...

Welche Sprachen verwenden Snake Case ?

Camel Case

``` var wifiPreSharedKey = "Hello"; // Functions function arrayMerge(arr1, arr2) { ... } // Classes mostly begin with upper case class LibraryCard { ... } ```

z.B. Java, Swift, JavaScript, Rust, PHP, Ruby...

Einige Sprachen verwenden beides Konventionen. z.B. Variablen in camel-case, Methoden in snake-case.

Camel Case oder Snake Case?
